## 摘要

本报告详细阐述了在 Laterre (2025) 博士论文研究框架基础上，面向数据中心废热回收场景所开展的建模、优化与动态性能分析工作。核心贡献包括三个创新点：①验证了 Lorenz 效率解耦框架在废热温度动态波动条件下的有效性；②建立了"废热温度→最优卡诺电池构型"的定量映射关系；③将部分负荷效率修正模型集成至逐时仿真框架，修正了原论文的乐观假设。所有建模工作基于 CBSim_clean 代码框架扩展实现。

## 1. 研究背景与问题定位

### 1.1 Laterre (2025) 框架的核心贡献与局限

Laterre (2025) 博士论文建立了卡诺电池系统的完整 4E 分析框架，其核心技术路线可概括为以下三层解耦架构：

**第一层（热力学优化层）**：对 16 种构型 × 296 个工况运行 NSGA-II 三目标优化，产生 Pareto 最优解集，并从中提取 4 个接口参数：

$$\mathcal{I} = \left\{ \Psi^{Lorenz}_{HP} = 0.50,\ \Psi^{Lorenz}_{HE} = 0.45,\ T^{ht}_{TES} = 95°C,\ T^{lt}_{TES} = 65°C \right\}$$

**第二层（容量设计层）**：以接口参数为约束，对系统容量变量 $\{P_{HP}, P_{ORC}, E_{TES}, P_{PV}\}$ 进行双目标优化（SSR vs IRR）。

**第三层（动态仿真层）**：以接口参数和容量参数为输入，进行 8760 步逐时仿真，计算年度 KPI。

**原论文的三个关键假设（局限性）**：

| 假设 | 原文描述 | 影响量级 |
|------|------|------|
| 恒温 TES 假设 | 储罐温度恒定为 95/65°C | 年度 RTE 高估约 5-7% |
| 额定工况假设 | HP 和 ORC 始终在额定工况运行 | 年度 RTE 高估约 5-10% |
| 完美预见假设 | 调度优化已知全年负荷和气象 | 实际性能低约 15-20% |

### 1.2 数据中心场景的特殊性

相比原论文的通用工况分析，数据中心废热回收场景具有以下特殊性：

1. **废热温度范围明确**：风冷（25-47°C）→ 冷板液冷（40-60°C）→ 高性能液冷（65-80°C），覆盖了 Laterre 论文揭示的 30K 临界点两侧。

2. **废热温度动态波动**：IT 负载波动导致废热温度在基准值附近 ±5-10°C 波动，这是 Laterre 论文中未考虑的动态效应。

3. **热电联产商业模式**：数据中心同时有供热（区域供热/建筑供暖）和供电需求，热售收益是 IRR 的关键驱动因素（Poletto et al., 2025 的 Sobol 指数分析表明热售价格的 Sobol 指数 > 0.4）。

| DC-C | 冷板液冷（冬季） | 50 | 5 | 45 | 临界点以上 |

**工作点设计逻辑**：通过在 30K 临界点两侧对称布置工作点，可以系统验证 Laterre 论文揭示的"临界点规律"在数据中心场景的适用性，并量化废热温度对最优构型选择的影响。

### 2.2 与 CBSim_clean 的集成方式

在 CBSim_clean 的 RHEIA case_description.py 中，新增 6 个工作点的配置文件：

```python
FIXED_PARAMS = {
    'T_hs_in': 50 + 273.15,   # 废热进口温度 [K]
    'T_hs_out': 40 + 273.15,  # 废热出口温度 [K]（假设10°C温降）
    'T_cs_in': 5 + 273.15,    # 冷源进口温度 [K]（冬季室外温度）
    'T_cs_out': 15 + 273.15,  # 冷源出口温度 [K]
    'W_net_HP': 100e3,         # HP 额定功率 [W]
}

# 废热温度动态波动模型（新增）
def get_dynamic_T_hs(T_hs_base, f_IT_load):
    """
    废热温度随 IT 负载动态变化
    T_hs_base: 基准废热温度 [°C]
    f_IT_load: IT 负载率 [0, 1]
    返回: 实时废热温度 [°C]
    """
    alpha_load = 10.0  # 负载-温度耦合系数 [°C]
    return T_hs_base + alpha_load * (f_IT_load - 0.5)
```

## 3. 个人工作二：全局 Pareto 前沿合并与构型选择

### 3.1 数学框架

对 6 个工作点 × 4 种构型（SBVCHP+SBORC, SBVCHP+SRORC, SRVCHP+SBORC, SRVCHP+SRORC）分别运行 NSGA-II，产生 24 个 Pareto 前沿。

**全局非支配排序**：将所有 24 个 Pareto 前沿的解合并，进行全局非支配排序，产生全局 Pareto 前沿 $\mathcal{P}^*_{global}$：

$$\mathcal{P}^*_{global} = \text{NDS}\left(\bigcup_{i=1}^{6} \bigcup_{j=1}^{4} \mathcal{P}^*_{ij}\right)$$

其中 $\text{NDS}(\cdot)$ 为非支配排序算子。

**构型统计分析**：对全局 Pareto 前沿中的解按构型分类，计算各构型的占比：

$$r_{j} = \frac{|\{x \in \mathcal{P}^*_{global} : \text{config}(x) = j\}|}{|\mathcal{P}^*_{global}|}$$

**无量纲参数计算**：

$$\Delta T_{HP,rel} = \frac{T^{ht}_{TES} - T_{hs}}{T^{ht}_{TES} - T_{cs}}, \quad \Delta T^{sp}_{TES,rel} = \frac{T^{ht}_{TES} - T^{lt}_{TES}}{\Delta T_{ORC}}$$

### 3.2 预期结果与分析

基于 Laterre 论文的规律和 Zhou et al. (2025) 的数据中心专项研究，预期：

- **DC-A/B（$\Delta T < 30$ K）**：HP 全力提温，亚临界回热型（SRVCHP+SRORC）在效率维度占主导（预期占比 > 60%）
- **DC-C/D（$\Delta T \approx 30-45$ K）**：过渡区域，构型选择对废热温度最敏感
- **DC-E/F（$\Delta T > 50$ K）**：HP 几乎不做功，跨临界基本型（SBVCHP+SBORC）在密度维度占主导

**串行策略依据**：选定 DC-D（$T_{hs}=55°C, T_{cs}=25°C$）作为代表性工作点，在全局 Pareto 前沿中选取偏好子集（效率优先），固定构型和接口参数，进入动态仿真层。

## 4. 个人工作三：Lorenz 效率解耦框架的动态验证

### 4.1 解耦有效性的数学条件

Lorenz 效率解耦的核心假设是 $\Psi^{Lorenz}$ 为常数，即：

$$\Psi^{Lorenz}_{HP} = \frac{COP^{actual}_{HP}}{COP^{Lorenz}_{HP}(\bar{T}^{TES}_H, \bar{T}^{hs}_C)} = \text{const}$$

在废热温度动态波动条件下，$\bar{T}^{hs}_C$ 随时间变化，因此需要验证 $\Psi^{Lorenz}_{HP}$ 是否仍然近似为常数。

**验证方法**：对 DC-D 工作点，在 $T_{hs} \in [45°C, 65°C]$ 范围内（±10°C 波动），分别运行 CBSim 热力学模型，计算 $\Psi^{Lorenz}_{HP}(T_{hs})$：

```python
import numpy as np
from src._module_carnot_battery import CBSimModel

def compute_lorenz_efficiency_vs_T_hs(T_hs_range, T_cs, T_TES_ht, T_TES_lt, config):
    """
    计算 Lorenz 效率折算系数随废热温度的变化
    """
    psi_hp_list = []
    
    for T_hs in T_hs_range:
        # 运行 CBSim 热力学模型（在最优设计点）
        model = CBSimModel(config=config)
        result = model.evaluate({
            'T_hs_in': T_hs + 273.15,
            'T_cs_in': T_cs + 273.15,
            'T_TES_ht': T_TES_ht + 273.15,
            'T_TES_lt': T_TES_lt + 273.15,
        })
        
        COP_actual = result['COP_HP']
        
        # 计算 Lorenz 理想 COP
        T_bar_H = (T_TES_ht - T_TES_lt) / np.log((T_TES_ht + 273.15) / (T_TES_lt + 273.15))
        T_bar_C = (T_hs - T_cs) / np.log((T_hs + 273.15) / (T_cs + 273.15))
        COP_Lorenz = (T_bar_H + 273.15) / ((T_bar_H + 273.15) - (T_bar_C + 273.15))
        
        psi_hp = COP_actual / COP_Lorenz
        psi_hp_list.append(psi_hp)
    
    return np.array(psi_hp_list)

# 执行验证
T_hs_range = np.linspace(45, 65, 21)  # ±10°C 波动范围
psi_values = compute_lorenz_efficiency_vs_T_hs(
    T_hs_range, T_cs=25, T_TES_ht=95, T_TES_lt=65, config='SRVCHP+SRORC'
)

# 评估稳定性
psi_mean = np.mean(psi_values)
psi_std = np.std(psi_values)
psi_cv = psi_std / psi_mean  # 变异系数

print(f"Ψ_HP 均值: {psi_mean:.4f}")
print(f"Ψ_HP 标准差: {psi_std:.4f}")
print(f"变异系数 CV: {psi_cv:.4f} ({psi_cv*100:.2f}%)")
```

**预期结果**：$\Psi^{Lorenz}_{HP}$ 的变异系数 CV < 3%，验证解耦框架在 ±10°C 废热温度波动下仍然有效。

### 4.2 创新点阐述

**创新点1的学术价值**：Laterre (2025) 仅在单一工况下计算了 $\Psi^{Lorenz}$ 并将其作为常数使用，但未验证其在废热温度波动条件下的稳定性。本工作通过系统的参数扫描，建立了 $\Psi^{Lorenz}(T_{hs})$ 的函数关系，并给出了解耦有效性的定量判据（CV < 5% 时可视为常数）。这一验证对于将解耦框架推广至实际工程应用具有重要意义。

## 5. 个人工作四：含废热温度动态的 8760 步逐时仿真框架

### 5.1 系统模型

**状态方程**：TES 储能状态（SOC）的差分方程：

$$E_{TES}[t+1] = E_{TES}[t] \cdot (1 - \sigma_{loss}) + COP_{HP}[t] \cdot W_{HP}[t] \cdot \Delta t - \frac{W_{ORC}[t]}{\eta_{ORC}[t]} \cdot \Delta t$$

其中 $\sigma_{loss}$ 为 TES 每小时热损失率（典型值 0.1%/h），$\Delta t = 1$ h。

**动态 COP 计算**（含废热温度动态）：

$$COP_{HP}[t] = \Psi^{Lorenz}_{HP} \cdot \frac{\bar{T}_{TES,H}}{\bar{T}_{TES,H} - \bar{T}_{hs}(T_{hs}[t])}$$

$$\bar{T}_{hs}[t] = \frac{T_{hs}[t] - T_{cs}[t]}{\ln\left(\frac{T_{hs}[t] + 273.15}{T_{cs}[t] + 273.15}\right)}$$

**废热温度动态模型**：

$$T_{hs}[t] = T_{hs,base} + \alpha_{load} \cdot \left(f_{IT}[t] - 0.5\right)$$

其中 $\alpha_{load} = 10°C$（负载-温度耦合系数，基于 Zhou et al. 2025 的数据）。

**ORC 效率动态计算**：

$$\eta_{ORC}[t] = \Psi^{Lorenz}_{HE} \cdot \left(1 - \frac{\bar{T}_{cs}[t]}{\bar{T}_{TES,H}}\right)$$

### 5.2 部分负荷效率修正（创新点3）

**部分负荷效率修正模型**（基于 Nadalon et al. 2023）：

$$COP_{HP,PL}[t] = COP_{HP,rated}[t] \cdot f_{PL,HP}(\alpha_{HP}[t])$$

$$\eta_{ORC,PL}[t] = \eta_{ORC,rated}[t] \cdot f_{PL,ORC}(\alpha_{ORC}[t])$$

其中部分负荷修正函数采用二次多项式拟合（基于 Nadalon et al. 2023 的数据）：

$$f_{PL,HP}(\alpha) = -0.15\alpha^2 + 0.30\alpha + 0.85, \quad \alpha \in [0.3, 1.0]$$

$$f_{PL,ORC}(\alpha) = -0.20\alpha^2 + 0.35\alpha + 0.85, \quad \alpha \in [0.3, 1.0]$$

其中 $\alpha = P_{actual}/P_{rated}$ 为负载率。

**创新点3的学术价值**：Laterre (2025) 明确承认"额定工况假设是非常乐观的"，但未提供修正方法。本工作将 Nadalon et al. (2023) 的部分负荷特性曲线集成至仿真框架，量化了部分负荷效率损失对年度 RTE 的影响，修正了原论文的系统性高估。

### 5.3 控制策略

采用基于规则的控制（RBC）策略：

```
优先级1（充电）：若 E_TES < 0.9 × E_TES_max 且 P_PV > P_load：
    W_HP = min(P_HP_rated, P_PV - P_load)
    
优先级2（放电）：若 E_TES > 0.2 × E_TES_max 且 P_load > P_PV：
    W_ORC = min(P_ORC_rated, P_load - P_PV)
    
优先级3（余量处理）：
    若 W_HP 仍有余量：购网补充
    若 W_ORC 仍有余量：售网
```

### 5.4 年度 KPI 计算

$$RTE = \frac{\sum_{t=1}^{8760} W_{ORC}[t]}{\sum_{t=1}^{8760} W_{HP}[t]}$$

$$SSR = \frac{\sum_{t=1}^{8760} \min(W_{ORC}[t] + P_{PV}[t],\ P_{IT}[t])}{\sum_{t=1}^{8760} P_{IT}[t]}$$

$$WHR = \frac{\sum_{t=1}^{8760} Q_{waste,recovered}[t]}{\sum_{t=1}^{8760} Q_{waste,total}[t]}$$

$$IRR: \quad \sum_{t=0}^{N} \frac{CF_t}{(1+IRR)^t} = 0$$

## 6. 创新点汇总与学术贡献

### 创新点1：Lorenz 效率解耦框架在动态废热条件下的有效性验证

**学术贡献**：首次系统验证了 Lorenz 效率折算系数在废热温度动态波动（±10°C）条件下的稳定性，建立了解耦有效性的定量判据，为将解耦框架推广至实际工程应用提供了理论依据。

**预期结论**：在 DC 场景的废热温度波动范围内，$\Psi^{Lorenz}_{HP}$ 的变异系数 CV < 3%，解耦框架有效。

### 创新点2：数据中心废热温度-最优构型定量映射

**学术贡献**：建立了从废热温度（35-80°C）到最优卡诺电池构型的定量映射关系，揭示了 30K 临界点在 DC 场景中的具体表现形式，为不同冷却技术的数据中心选择最优卡诺电池构型提供了直接的工程指导。

**预期结论**：
- $T_{hs} < 45°C$（风冷）：亚临界回热型（SRVCHP+SRORC）最优
- $T_{hs} \in [45°C, 60°C]$（冷板液冷）：构型选择对废热温度敏感，需按具体工况优化
- $T_{hs} > 60°C$（高性能液冷）：跨临界基本型（SBVCHP+SBORC）在密度维度最优

### 创新点3：含部分负荷修正的年度性能评估

**学术贡献**：将部分负荷效率修正模型集成至 8760 步逐时仿真框架，首次在数据中心场景下量化了部分负荷效率损失对年度 RTE 的影响，修正了 Laterre (2025) 的系统性高估。

**预期结论**：考虑部分负荷效率后，年度 RTE 比额定工况假设低约 5-8%；SSR 比额定工况假设低约 3-5%。
